## In Python, there are some difference on `if` statmement.

### Difference of `is` and `==`:
`==` compares the value while `is` compares the address of the value.

```
Example:
>>> a = [12]
>>> b = [12]
>>> id(a)
4327524472
>>> id(b)
4327570104
```
